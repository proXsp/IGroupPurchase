# IGroupPurchase

目标为打造支持多端的社区团购系统，服务端基于spring cloud alibaba微服务架构，前端vue、flutter等。

## 模块
igp.base -- 基础模块 <br>
igp.core -- 核心模块 <br>
igp.security -- 安全模块 <br>
igp.user -- 用户模块 <br>
igp.login -- 登录模块 <br>
igp.flow -- 流程模块 <br>
igp.group -- 团模块 <br>
igp.goods -- 商品模块 <br>
igp.search -- 搜索模块 <br>
igp.evaluation -- 评价模块 <br>
igp.message -- 消息模块 <br>
igp.cc -- 通讯模块 <br>
igp.recommend -- 推荐模块 <br>
igp.ad -- 广告模块 <br>
igp.pay -- 支付模块 <br>
igp.community -- 社区模块 <br>

## 基础设施
注册中心、配置中心：nacos2.0 <br>
网关：spring cloud gateway <br>
负载均衡：ribbon <br>
稳定性平台：sentinel <br>
链路追踪：zipkin <br>
缓存中间件：redis <br>
消息中间件：rocket mq <br>
RPC：dubbo <br>

## 参与
使用p3c code style
